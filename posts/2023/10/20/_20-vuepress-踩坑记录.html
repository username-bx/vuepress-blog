# vuepress踩坑记录

  1. 部署到服务器,样式丢失

    // .vuepress/config.js
// 添加字段 base
// 在本地开发阶段,需要删掉base属性. 只有build时,需要添加
base: './'

  2. 部署到服务器上,页面一刷新,样式丢失

    1. 文件路径: node_modules/@vuepress/core/lib/client/app.js
    2. 注释 router 对象 mode

  3. 使用yarn安装插件后,@vuepress包会重新下载

  4. 注释 router 对象 mode,不然会出现样式丢失问题.

  5. 自定义页面,部署后,点击报错

    1. 注释 router 对象 mode,不然会出现样式丢失问题.
